# 01-IFC_structure

Simple workspace for IFC sample files and related scripts.

## Contents

- `IFC_files/`: sample IFC models
- `.env`: local environment variables
- `requirements.txt`: Python dependencies

## Getting Started

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Add your local configuration in `.env`.
